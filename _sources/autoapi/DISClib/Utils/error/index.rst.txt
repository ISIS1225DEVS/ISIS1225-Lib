.. py:function:: error_handler(context: str, func_name: str, err: Exception) -> None

   *error_handler()* recibe el contexto, nombre de la función y la excepción para lanzar un mensaje de error detallado y el traceback.

   :param context: nombre del contexto donde ocurrió el error.
   :type context: str
   :param func_name: nombre de la función donde ocurrió el error.
   :type func_name: str
   :param err: excepción lanzada.
   :type err: Exception

   :raises type: excepción con el mensaje de error detallado y el traceback.
