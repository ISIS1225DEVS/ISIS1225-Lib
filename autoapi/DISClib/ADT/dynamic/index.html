DISClib.ADT.dynamic

Esta clase permite importar dinámicamente módulos y clases de módulos dentro de DISClib.

IMPORTANTE: Este código y sus especificaciones para Python están basados en las implementaciones propuestas por los siguientes autores/libros:

  1. Algorithms, 4th Edition, Robert Sedgewick y Kevin Wayne.

  2. Data Structure and Algorithms in Python, M.T. Goodrich, R. Tamassia, M.H. Goldwasser.

DynamicImporter

DynamicImporter permite importar dinámicamente módulos y clases de módulos según la configuración de un archivo JSON y las especificaciones del usuario.
class DISClib.ADT.dynamic.DynamicImporter(implementation: str, package: str, **kwargs)[fuente]

DynamicImporter permite importar dinámicamente módulos y clases de módulos según la configuración de un archivo JSON y las especificaciones del usuario.

Muestra:

ValueError – no se puede importar el módulo especificado.

Devuelve:

instancia de la clase dinámica.

Tipo del valor devuelto:

DynamicImporter
