AdjacencyMatrix

class Src.DISClib.DataStructures.adjmatrix.AdjacencyMatrix[fuente]

Bases: Generic[Src.DISClib.Utils.default.T]

AdjacencyMatrix representa la estructura de datos para arreglos dinamicos (Array List), Implementada con Generic[T] y @dataclass para que sea una estructura de datos genérica.

Parámetros:

Generic (T) – TAD (Tipo Abstracto de Datos) o ADT (Abstract Data Type) para una estructura de datos genéricas en python.

Devuelve:

ADT de tipo AdjacencyMatrix o Arreglo Dinámico.

Tipo del valor devuelto:

AdjacencyMatrix

iodata: List[Src.DISClib.Utils.default.T] | None[fuente]

Lista nativa de Python personalizable por el usuario para inicializar la estructura. Por defecto es None y el usuario puede incluirla como argumento al crear la estructura.

cmp_function: Callable[[Src.DISClib.Utils.default.T, Src.DISClib.Utils.default.T], int] | None[fuente]

Función de comparación personalizable por el usuario para reconocer los elementos dentro de la estructura. Es un argumento configurable al crear la estructura. Por defecto es la función lt_default_cmp_funcion() propia de DISClib.

elements: List[Src.DISClib.Utils.default.T][fuente]

Lista nativa de Python que contiene los elementos de la estructura.

key: str | None[fuente]

Nombre de la llave opcional que se utiliza para comparar los elementos del AdjacencyList, Por defecto es None y el __post_init__() configura la llave por defecto la llave “id” en DEFAULT_DICT_KEY.

_size: int = 0[fuente]

Es el número de elementos que contiene la estructura, por defecto es 0 y se actualiza con cada operación que modifica la estructura.

__post_init__() → None[fuente]

__post_init__() configura los valores por defecto para la llave (“key”) y la función de comparación (“cmp_function”). Si el usuario incluye una lista nativa de python como argumento, se agrega a la lista de elementos del AdjacencyMatrix.

default_cmp_function(elm1, elm2) → int[fuente]

default_cmp_function() procesa con algoritmica por defecto la lista de elementos que procesa el AdjacencyMatrix. Es una función crucial para que la estructura de datos funcione correctamente.

Parámetros:

  • elm1 (Any) – primer elemento a comparar.

  • elm2 (Any) – segundo elemento a comparar.

Devuelve:

respuesta de la comparación entre los elementos, 0 si son iguales, 1 si elm1 es mayor que elm2, -1 si elm1 es menor.

Tipo del valor devuelto:

int

_handle_error(err: Exception) → None[fuente]

_handle_error() función privada que maneja los errores que se pueden presentar en el AdjacencyMatrix.

Si se presenta un error en AdjacencyMatrix, se formatea el error según el contexto (paquete/clase), la función que lo generó y lo reenvia al componente superior en la jerarquía de llamados para manejarlo segun se considere conveniente.

Parámetros:

err (Exception) – Excepción que se generó en el AdjacencyMatrix.

_check_type(element: Src.DISClib.Utils.default.T) → bool[fuente]

_check_type() función privada que verifica que el tipo de dato del elemento que se quiere agregar al AdjacencyMatrix sea del mismo tipo contenido dentro de los elementos del AdjacencyMatrix.

Muestra:

TypeError – error si el tipo de dato del elemento que se quiere agregar no es el mismo que el tipo de dato de los elementos que ya contiene el AdjacencyMatrix.

Parámetros:

element (T) – elemento que se quiere procesar en AdjacencyMatrix.

Devuelve:

operador que indica si el ADT AdjacencyMatrix es del mismo tipo que el elemento que se quiere procesar.

Tipo del valor devuelto:

bool
